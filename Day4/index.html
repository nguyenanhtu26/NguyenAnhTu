CSS là viết tắt của Cascading Style Sheets

CSS mô tả các thành phần trong html được hiện thị trên màn hình và trên các thiết bị khác

CSS tiết khiệm thời gian làm việc

Nó có thể kiểm soát bố cục rất nhiều trang web trên cùng một thời điểm

CSS được thêm và html có 3 cách

Cách 1 gọi là inline css sử dụng thuộc tính style trong thẻ html

Cách 2 gọi là internal sử dụng thẻ style trong thẻ head

Cách 3 gọi là external sử dụng một file css bên ngoài html
